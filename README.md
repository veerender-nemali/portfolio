# Portfolio

A clean and modern portfolio built with Next.js and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.

## Features

- Responsive design
- Dark mode support
- Custom cursor
- Smooth animations
- Theme switcher

## Usage

Feel free to use this portfolio as a template for your own. Clone it, customize the content, and make it yours!

Edit the components in the `components/` folder and update your information in `app/page.js`.
